# RunPaceFlow UI 优化计划

## 优化目标

参考 Strava、Nike Run Club、Garmin Connect 等现代跑步应用，提升数据可视化和用户体验。

- [x] 在 ActivityCard 中添加平均配速显示

- [x] 日期显示添加星期几
